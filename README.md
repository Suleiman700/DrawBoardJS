# DrawBoardJS
A drawboard built in JavaScript

Demo: https://heroic-croquembouche-3c434b.netlify.app/

---

Features:
* Responsive.
* Multiple colors.
* Undo/Redo/Erase buttons.
* Changeable pen size.
